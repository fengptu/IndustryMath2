ที่มาของสูตรค่าเฉลี่ยของฟังก์ชัน (Average Value of a Function)

สูตรค่าเฉลี่ยของฟังก์ชัน คือ $f_{avg} = \frac{1}{b-a}\int_{a}^{b}f(x)dx$ มีที่มาจากแนวคิดเดียวกับการหาค่าเฉลี่ยเลขคณิตธรรมดา แต่ขยายให้ใช้กับฟังก์ชันต่อเนื่องได้

แนวคิดเบื้องต้น

การหาค่าเฉลี่ยของชุดข้อมูลตัวเลข เช่น ${y_1, y_2, ..., y_n}$ เราจะใช้สูตร $\text{Average} = \frac{y_1 + y_2 + ... + y_n}{n}$ โดย $n$ คือจำนวนข้อมูลทั้งหมด

เมื่อนำแนวคิดนี้มาใช้กับฟังก์ชันต่อเนื่อง $f(x)$ บนช่วง $[a, b]$ จะเกิดปัญหาเพราะฟังก์ชันมีค่ามากมายนับไม่ถ้วนตลอดช่วง การจะบวกค่าทั้งหมดเข้าด้วยกันจึงเป็นไปไม่ได้ ดังนั้นเราจึงต้องใช้ การอินทิเกรต ซึ่งเปรียบเสมือนการ "บวก" ค่าของฟังก์ชันตลอดช่วงต่อเนื่อง

ที่มาของสูตร

  1. แบ่งช่วง: แบ่งช่วง $[a, b]$ ออกเป็น $n$ ส่วนย่อยเท่าๆ กัน โดยแต่ละส่วนมีความกว้าง $\Delta x = \frac{b-a}{n}$
  2. สุ่มตัวอย่าง: เลือกจุดตัวอย่าง $x_i^*$ หนึ่งจุดในแต่ละส่วนย่อย แล้วหาค่าของฟังก์ชันที่จุดนั้น คือ $f(x_i^*)$
  3. หาค่าเฉลี่ยโดยประมาณ: ค่าเฉลี่ยของฟังก์ชันโดยประมาณ (จากการสุ่มตัวอย่าง) คือ
    Average ≈ (f(x*₁) + f(x*₂) + ... + f(x*ₙ)) / n
  4. ปรับรูปสมการ: เราทราบว่า $n = \frac{b-a}{\Delta x}$ ดังนั้นเราสามารถจัดรูปสมการใหม่ได้เป็น
    Average ≈ (1 / (b-a)) * Σ[i=1 to n] f(x*ᵢ)Δx

บทสรุป

ในสมการสุดท้าย $\sum_{i=1}^{n} f(x_i^*)\Delta x$ คือ ผลบวกรีมันน์ (Riemann Sum) ซึ่งเป็นการประมาณค่าของการอินทิเกรต ยิ่งเราแบ่งช่วงย่อยมากขึ้น ($n \rightarrow \infty$) ค่าประมาณนี้ก็จะยิ่งเข้าใกล้ค่าจริงของการอินทิเกรต

เมื่อ $n$ เข้าสู่ค่าอนันต์ ผลรวมรีมันน์ก็จะกลายเป็นการอินทิเกรต:

lim[n→∞] Σ[i=1 to n] f(x*ᵢ)Δx = ∫[a to b] f(x)dx

เมื่อนำมาแทนที่ในสมการค่าเฉลี่ย เราจะได้สูตรที่สมบูรณ์:

$f_{avg} = \frac{1}{b-a}\int_{a}^{b}f(x)dx$

กล่าวโดยสรุป การอินทิเกรต $\int_{a}^{b}f(x)dx$ คือ "ผลรวม" ของค่าฟังก์ชัน ทั้งหมด และเราหารด้วยความยาวของช่วง $(b-a)$ เพื่อหา ค่าเฉลี่ย ที่แท้จริงของฟังก์ชันบนช่วงนั้นๆ
